Understanding User Personas in Brown University's Laundry Service Interface

User-friendly interfaces are crucial for everyday campus services, yet the experience of students using Brown University’s laundry machines often goes unexamined.

This case study explores the personas of users interacting with the university’s laundry service interface. Understanding these personas will help inform design recommendations for a more intuitive and accessible laundry experience at Brown.

The Laundry Service Kiosk Interface:

The laundry machine kiosk interface allows users to see what machines are available for use, and to select and swipe for (using their student ID) the machine they would like to use.

Key Components:
  • Main touchscreen
  • Card reader touchscreen
  • Card reader

User Behavior

Through direct observation of users interacting with the laundry kiosk interface, several key patterns and challenges emerged.

Observations
Quick Button Selection Users often select buttons rapidly without thinking, leading to mistakes and backtracking.
Confusion Between Kiosk & Machines Some users repeatedly walk back and forth between the kiosk and washers/dryers, struggling to match selections.
Card Swiping Issues Many users swipe their student ID incorrectly, requiring multiple attempts before successful payment.
Wrong Machine Selection Users sometimes select the wrong machine, forcing them to go back multiple steps to remove it from their cart.
Settings Adjustment Confusion Users attempt to edit machine settings via the kiosk before realizing settings must be changed on the actual machine.

User Interviews

To gain deeper insights into the user experience with the laundry kiosk interface, I conducted interviews with users who regularly interact with the system.

Question
How would you describe your overall satisfaction with the laundry machine reader? Users found the system generally straightforward but noted that it can be faulty and sometimes inaccurately displays working washers.
Were there any moments of confusion or hesitation while using the interface? If so, can you describe them? Most users found the interface intuitive, but some were confused by the inability to change washer and dryer settings from the kiosk.
What features, if any, do you find particularly intuitive? The clear machine selection process (with large, visible numbers) was praised. The payment system was also seen as quick and easy.
If you could change one thing about the interface, what would it be? The most requested feature was the ability to start washers from the kiosk instead of having to go back to the machines. Users also wanted washer settings to be adjustable from the kiosk.

Conclusion

Brown's laundry kiosk interface falls short in terms of difficulty in selecting the correct machine, the inability to adjust settings from the kiosk, and confusion when swiping ID cards. My findings suggest that allowing users to start machines from the interface and improving feedback for card swipes could significantly improve the experience.
